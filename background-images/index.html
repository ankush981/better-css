Blend Mode

One of the really poorly known but magnificent properties is the blend mode for background images. It allows us to get creative and produce nice effects by "mixing" the background image colors with other backgrounds present.

Hover effects

One of the cool effects you can create on a background image is to make it come alive (in some way) on hover. One way you can do this is by adding a translate effect and making the background image bigger on hover (yes, background-size can take values such as 150%.).

HOWEVER: For this to work, we have to set the background-size to 100%. It has the same effect as cover, but the latter interferes with the transition and makes it jerky.
